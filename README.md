I am a dedicated **Artificial Intelligence and Machine Learning Enthusiast** with a strong foundation in **Python** and **C++**. My technical journey is driven by a fascination with how data and algorithms can solve complex, real-world problems. I spend my days working in VS Code, constantly expanding my knowledge of modern ML libraries and frameworks to build efficient, scalable models.

*   👨‍💻 **Workflow:** My daily development environment is heavily centered around **VS Code**, customized and optimized for Python and C++ development.
